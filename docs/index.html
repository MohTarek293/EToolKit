Revit Add-in

Overview

EToolKit is a productivity add-in for Autodesk Revit designed to help teams validate common issues, improve documentation quality, and automate repetitive drafting/model checks.

This site provides installation guidance, feature summaries, and support/contact details for Autodesk App Store review and end users.

Compatibility

  • Supported Revit versions: 2022 – 2024
  • Platform: Windows

If you require support for another version, please contact support.

Installation

  1. Download and run the EToolKit installer.
  2. Follow the on-screen installation steps.
  3. Launch Revit → open the Add-ins tab → locate EToolKit.

EToolKit loads automatically once installed.

Getting Started

  1. Open any Revit project.
  2. Go to Add-ins → EToolKit.
  3. Select a tool from the EToolKit panel and follow the prompts.

Tip: For best results, save your project before running validation or batch operations.

Tools Included

1) Circuit Checker

Purpose: Validates electrical circuits in the current project to ensure they are consistently configured.

Where to find it: Add-ins → EToolKit → Circuit Checker.

2) Tags & Dimensions Checker

Purpose: Helps identify common documentation issues related to tags and dimensions.

Where to find it: Add-ins → EToolKit → Tags & Dimensions Checker.

3) Fillet Lines

Purpose: Creates fillet lines to improve clarity and presentation in drawings.

Where to find it: Add-ins → EToolKit → Fillet Lines.

4) Host Wizard

Purpose: Assists with managing and resolving host-related issues for elements.

Where to find it: Add-ins → EToolKit → Host Wizard.

5) Levels Fix

Purpose: Helps identify and correct common inconsistencies in level assignments.

Where to find it: Add-ins → EToolKit → Levels Fix.

6) Renumbering

Purpose: Renumbers selected elements according to user-defined or standard rules.

Where to find it: Add-ins → EToolKit → Renumbering.
